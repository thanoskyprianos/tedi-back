1. Chat folder
Chat folder contains classes that handle messages.
This includes: sending, receiving, processing messages.

ChatMessageController: handles & processes incoming messages, sends notifications.
ChatMessageService: Saving and retrieving chat messages.

2. Chat Room folder
Chat Room folder contains classes that handle chat rooms.
This includes creation, retrieval and management of chat rooms.

ChatRoom: Representation of a chat room entity.
ChatRoomRepo: Repository for chat room entities.
ChatRoomService: Create and retrieve chat rooms.

3. ws config folder
It includes classes that set up and configure the operations of a WebSocket.
